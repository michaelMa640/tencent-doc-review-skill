# 基于LLM的文章审核批注工具 (tencent-doc-review)

## 项目概述

**项目名称**: tencent-doc-review  
**项目路径**: `E:\VibeCoding\tencent-doc-review`  
**开发语言**: Python 3.10+  
**核心功能**: 基于大语言模型的文章审核、批注和质量评估

### 核心能力

1. **事实核查** - 自动识别并验证文章中的事实性陈述
2. **结构匹配** - 对比文档结构与模板，识别缺失/多余章节
3. **质量评估** - 多维度评估文档质量（完整性、逻辑性、论证质量等）
4. **腾讯文档集成** - 直接读取文档内容，自动添加批注

### 核心模块说明

#### 1. FactChecker (事实核查器)
- **文件**: `fact_checker.py` (~783行)
- **功能**: 提取声明、验证事实、批量处理
- **输出**: `FactCheckResult` 列表

#### 2. StructureMatcher (结构匹配器)
- **文件**: `structure_matcher.py` (~650行)
- **功能**: 解析结构、匹配章节、检测缺失/多余
- **输出**: `StructureMatchResult`

#### 3. QualityEvaluator (质量评估器)
- **文件**: `quality_evaluator.py` (~600行)
- **功能**: 7维度评估、质量等级、改进建议
- **输出**: `QualityReport`

#### 4. DocumentAnalyzer (文档分析器主类)
- **文件**: `document_analyzer.py` (~2200行)
- **功能**: 整合三个核心模块、提供统一接口、批量处理
- **输出**: `AnalysisResult`

## 使用说明

### 快速开始

1. **安装依赖**
   ```bash
   pip install -r src/requirements.txt
   ```

2. **配置 API 密钥**
   ```python
   # 在 config/config.yaml 中配置
   deepseek:
     api_key: "your-api-key"
     api_url: "https://api.deepseek.com"
   ```

3. **运行分析**
   ```python
   from tencent_doc_review.analyzer.document_analyzer import DocumentAnalyzer
   
   analyzer = DocumentAnalyzer(deepseek_client)
   result = await analyzer.analyze(document_text)
   ```
