# 项目方案

## 项目名称

Tencent Doc Review Skill

## 项目目标

构建一个基于 LLM 的腾讯文档审核工具，用于:

- 读取腾讯文档内容
- 按模板检查结构完整性
- 进行事实核查和质量评估
- 生成结构化审核结果
- 以报告或固定区块写回方式交付审核意见

## 当前架构

```text
src/tencent_doc_review/
  cli.py
  config.py
  deepseek_client.py
  tencent_doc_client.py
  mcp_client.py
  analyzer/
```

## 设计原则

- 接入层与分析层解耦
- 不把未验证的原生批注能力作为上线前提
- 优先保证“可读取、可分析、可交付”
- 写回能力采用逐步增强策略

## 本期重点

1. 打通 DeepSeek 和分析器链路
2. 打通腾讯文档读取链路
3. 统一报告输出
4. 保持 CLI 和打包配置可用
