#### 3.4 MCP 客户端 (`src/tencent_doc_review/mcp_client.py`)

**文件路径**: `文稿/tencent-doc-review/src/tencent_doc_review/mcp_client.py`  
**代码行数**: ~700 行  
**状态**: ✅ 已完成

**核心功能**:
- 封装腾讯文档 MCP 协议调用
- 支持通过 `mcporter` CLI 调用
- 提供文档读取、批注插入等功能

**核心类**:

```python
class TencentDocMCPClient:
    """腾讯文档 MCP 客户端"""
    
    # 文档操作
    async def list_documents(folder_id: str) -> List[DocumentInfo]
    async def get_document_content(file_id: str) -> str
    
    # 批注操作
    async def add_comment(file_id: str, comment: Comment) -> bool
    async def add_comments_batch(file_id: str, comments: List[Comment]) -> Dict[str, Any]
```

**数据模型**:

```python
@dataclass
class DocumentInfo:
    file_id: str
    title: str
    doc_type: str
    create_time: str
    modify_time: str
    url: Optional[str] = None

@dataclass
class Comment:
    content: str
    position: Dict[str, Any]
    quote_text: Optional[str] = None
    comment_type: str = "text"
```

**使用示例**:

```python
# 初始化客户端
client = TencentDocMCPClient()

# 获取文档列表
docs = await client.list_documents(folder_id="your_folder_id")

# 获取文档内容
content = await client.get_document_content(file_id="your_doc_id")

# 添加批注
comment = Comment(
    content="此处数据需要核实",
    position={"paragraph": 5, "offset": 10},
    quote_text="2023年GDP增长5.2%",
    comment_type="warning"
)
success = await client.add_comment(file_id="your_doc_id", comment=comment)

# 关闭客户端
await client.close()
```
